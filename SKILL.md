# 赚钱灵感生成器

自动发现 AI 变现机会，生成可落地的赚钱灵感。

## 核心功能

### 1. 热门项目监控
- 监控 GitHub Trending AI 项目
- 分析新增星标速度
- 识别热门技术趋势

### 2. 变现潜力分析
- 判断项目是否"能赚钱"
- 分析市场需求和竞争
- 给出潜力评分（高/中/低）

### 3. 灵感生成
- 自动生成赚钱灵感
- 提供具体实现路径
- 估算启动成本和预期收入

### 4. 个性化推荐
- 根据用户偏好推荐
- 按领域分类（AI 工具、SaaS、内容创作）
- 按难度分级（入门/进阶/专业）

## 变现潜力判断规则

### 高潜力特征
- ✅ 解决明确痛点
- ✅ 技术门槛适中
- ✅ 有付费用户验证
- ✅ 可规模化复制
- ✅ 新增星标 > 100/周

### 低潜力特征
- ❌ 技术门槛太高
- ❌ 市场太小众
- ❌ 已有巨头垄断
- ❌ 需要大量资金
- ❌ 无明确商业模式

## 配置

在 `config.py` 中配置：
- `GITHUB_TOKEN`: GitHub API Token（可选，提高速率限制）
- `MIN_STARS`: 最小星标数（默认 100）
- `CATEGORIES`: 关注的领域（默认 AI/LLM/Agent）
